2nd QUARTER LESSONS





LESSON 1

In the first lesson, we learned about CSS: its definition, benefits in using, and the three layers, including CSS, that make up a webpage. CSS helps style our webpages in ways that HTML and JavaScript aren't able to. With it, we can express our ideas and designs and do it in a way that is efficient and pleasing.

LESSON 2

In this lesson, we learned about the structure of CSS that holds up the document and makes it unique, and the kinds of CSS style sheets: Inline, External, and Internal. Each of them have their own advantages and disadvantages, leaving multiple options for all kinds of coding situations. We also learned how the structures of these style sheets and their difference with each other.

LESSON 3

The third lesson is all about block and inline-level elements, including the <div> and <span> tags. These act as the structure and style of the webpage, with block level elements stacking up on each other and taking the space, and inline-level styles the text in these sections.

LESSON 4

Lesson 4 is all about class and selectors, which play a huge role when you're coding in an external or internal style sheets. They ensure efficiency and consistency when styling an HTML document as you can apply it to any element that you want, and you can also apply multiple styles to one element.

LESSON 5

The fifth lesson is about the different types of selectors, being class, ID, element/tag, universal, group, and attribute. These make targeted styling even more specific, cleaning the overall look of the code and giving greater flexibility and control.

LESSON 6

This lesson is about controlling the dimensions and visibility of HTML elements using CSS, and also how to apply pseudo-classes to links. Properties like width, height, max-width, and visibility help manage appearance of the elements on a webpage. There are four pseudo-classes for links: ":link, :visited, :hover, and :active", which have an important order (LVHA).
